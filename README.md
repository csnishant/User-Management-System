# 🚀 User Management System (MERN Stack)

🔗 **Live Demo:** https://user-management-sys-nis.netlify.app/dashboard  

---

## 📌 Overview  
This is a full-stack **User Management System** built using the MERN stack.  
It provides secure authentication, role-based authorization (RBAC), and complete user lifecycle management with a clean and responsive UI.

The application supports multiple user roles and ensures controlled access based on permissions.

---

## ⚙️ Tech Stack  

### Frontend  
- React (Hooks)  
- Context API (State Management)  
- Framer Motion (Animations)  
- Responsive UI  

### Backend  
- Node.js  
- Express.js  

### Database  
- MongoDB  

### Authentication  
- JWT (JSON Web Token)  
- bcrypt (Password Hashing)  

---

## 🔐 Features  

### 🔑 Authentication  
- Secure login using JWT  
- Password hashing with bcrypt  
- Protected routes  

---

### 🛡️ Role-Based Access Control (RBAC)  

#### 👨‍💼 Admin  
- Full system access  
- Create, update, delete users  
- Assign roles  
- View all users  

#### 🧑‍💻 User  
- View own profile  
- Update personal details  
- Cannot access other users  

---

### 👥 User Management  
- Create users  
- Update user details  
- Soft delete / deactivate users  
- Search, filter, pagination  
- View user details  

---

### 📊 Audit & Tracking  
- createdAt / updatedAt  
- createdBy / updatedBy  
- Audit info visible in user details  

---

### 🎨 UI/UX  
- Fully responsive design  
- Smooth animations using Framer Motion  
- Role-based UI rendering
